Desde la App Asesor puedes generar una nueva cotización desde el botón flotante **+** disponible en la pantalla de inicio. Este flujo es igual al de escritorio, pero adaptado a dispositivos móviles.

Para consultar el proceso desde dashboard, ingresa
**[aquí](/plataforma-actual/crm_docs/crear-cotizacion)**

---

## Inicia una nueva cotización

<Card title="Acceso desde inicio" icon="plus">
  En la esquina inferior derecha de la pantalla de inicio, pulsa el botón
  flotante **+** y elige la opción **“Nueva Cotización”**.
</Card>

<Card title="Selecciona un prospecto" icon="user-plus">
  Puedes buscar un cliente existente o registrar uno nuevo. Luego selecciona la
  moneda: **MXN** o **USD**.
</Card>

---

## Define el inmueble

<Card title="Selecciona desarrollo, proyecto y etapa" icon="building">
  Elige en orden el desarrollo, luego el proyecto y después la etapa. Solo verás
  las opciones habilitadas según tu canal.
</Card>

<Card title="Selecciona un lote" icon="map">
  Elige el lote desde el mapa visual interactivo. Verde: Disponible · Amarillo:
  Apartado · Gris: No disponible.
</Card>

<Card title="Verifica detalles del lote" icon="file-lines">
  Superficie, precio por m², precio total y gastos administrativos se mostrarán
  precargados para tu validación.
</Card>

---

## Ajusta condiciones de pago

<Card title="Enganche y plazos" icon="sliders">
  Define el porcentaje de enganche y los meses de financiamiento utilizando los
  controles deslizables.
</Card>

<Card title="Aplicar promoción" icon="percent">
  Puedes seleccionar: - Descuento automático - Promoción activa - Código
  promocional - Ninguno
</Card>

---

## Revisión y creación

<Card title="Resumen final" icon="clipboard-list">
  Se muestra toda la información financiera generada: - Enganche - Precio de
  lista - Mensualidad - Apartado - Total financiado - Promoción aplicada
</Card>

<Card title="Consulta cuotas" icon="calendar">
  Puedes consultar el desglose completo de cuotas mensuales, incluyendo enganche
  y pagos futuros.
</Card>

<Card title="Generar cotización" icon="file-signature">
  Al presionar **Crear**, el sistema genera la cotización y te dirige a la
  pantalla de compartir.
</Card>

---

## Compartir enlace

<Card title="Enlace de onboarding" icon="link">
  Al finalizar, recibirás un enlace único que podrás: - Copiar al portapapeles -
  Compartir por WhatsApp directamente desde la App
</Card>

<Note>
  El cliente podrá continuar su proceso de compra desde este enlace mediante el
  flujo automatizado. Más información en: **[ventas
  automatizadas](/procesos-digitales/ventas-automatizadas)**
</Note>

## Preguntas frecuentes (FAQ)

<AccordionGroup type="multiple">
  <Accordion title="¿Puedo registrar un nuevo prospecto desde el flujo?">
    Sí. Justo al inicio del flujo puedes crear un nuevo prospecto si el cliente aún no está registrado.
  </Accordion>

{" "}

<Accordion title="¿Qué pasa si un lote ya no está disponible?">
  Se mostrará como no disponible en el mapa. Selecciona otro lote que aparezca
  en color verde.
</Accordion>

{" "}

<Accordion title="¿Puedo usar promociones activas desde la app?">
  Sí. Se cargan automáticamente o puedes ingresar un código si tienes uno
  disponible.
</Accordion>

{" "}

<Accordion title="¿Puedo modificar la cotización una vez creada?">
  No. Para modificarla deberás generar una nueva cotización.
</Accordion>

{" "}

<Accordion title="¿Cómo se comparte el enlace?">
  El sistema muestra el link con botón de copiar y compartir por WhatsApp.
  Puedes enviarlo al cliente fácilmente.
</Accordion>

  <Accordion title="¿El cliente puede continuar su proceso desde el link?">
    Sí. Este enlace inicia el flujo automatizado de apartado, documentación y firma.
  </Accordion>
</AccordionGroup>
