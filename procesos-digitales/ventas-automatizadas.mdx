<Note>
  Este proceso lo genera el cliente, por lo cual, un asesor previamente debió
  haber realizado la cotización
</Note>

Una vez que se ha generado una cotización para un prospecto, el sistema crea automáticamente un **enlace único** que será enviado por el asesor vía WhatsApp, correo electrónico u otro medio. Este enlace tiene una vigencia de **72 horas** y permite que el cliente continúe con su proceso de compra de forma digital.

---

## Proceso para el cliente

A continuación, se describen los pasos que el cliente deberá completar una vez que acceda al enlace generado por el sistema:

### 1. Pago de apartado

<Card>
  <b>Confirmación de monto y método de pago</b>
  <br />
  En esta pantalla se muestran los detalles de la cotización: monto total,
  enganche, mensualidades, así como el monto del apartado.
</Card>

- Verifica que el monto del apartado mostrado sea correcto.
- Ingresa tus datos correctamente, especialmente el correo electrónico.
- Asegúrate de que los datos de tu tarjeta sean válidos y cuentes con fondos suficientes.

<Note>
  Al completar este paso, aceptas los Términos y Condiciones del contrato.
</Note>

---

### 2. Documentación

<Card>
  <b>Formalización del contrato</b>
  <br />
  Una vez realizado el pago, deberás seleccionar el tipo de contrato y completar
  los datos requeridos.
</Card>

#### Selección de tipo de contrato

- **Contrato individual**: para una sola persona.
- **Contrato plural**: si hay más de un dueño, deberás agregar a cada socio y todos deberán llenar sus datos.

#### Documentos requeridos

- Selecciona con qué documento deseas identificarte: **INE** o **Pasaporte**.
- Si seleccionas **INE**, deberás subir imagen **frontal y reverso**, e ingresar la **clave de lector**.
- Si eliges **Pasaporte**, deberás subir la imagen **delantera** del documento y capturar el **número de pasaporte**.

<Note>
  Todos los campos marcados como obligatorios deben ser completados
  correctamente para continuar con el proceso.
</Note>
