## Mejoras en automatización, gestión de ventas y unificación de sistemas

### Dashboard

#### Validación de facturas - **feature**

<img
  className="rounded-xl"
  width="500"
  height="500"
  src="/images/factura.png"
/>

Se incorpora un nuevo módulo que permite a los asesores cargar sus facturas directamente en Dashboard para validar el cobro de sus comisiones. Este proceso sustituye el modelo manual y mejora el control fiscal.

##### Características principales:

- Validación automática del XML cargado por el asesor.
- Asociación del archivo con el RFC registrado en su perfil.
- Cálculo y confirmación del monto a facturar con base en el corte autorizado.
- Visualización clara de líneas pendientes de cobro por periodo.
- Validación en tiempo real con respuesta inmediata.

<Card title="Campos validados del CFDI" icon="file-lines" horizontal={true}>
  <ul>
    <li>RFC emisor y receptor</li>
    <li>Régimen fiscal</li>
    <li>Folio fiscal (UUID)</li>
    <li>Fecha y hora de emisión</li>
    <li>Uso del CFDI</li>
    <li>Moneda, método y forma de pago</li>
    <li>Claves SAT (producto/unidad)</li>
    <li>Subtotal, impuestos, total</li>
  </ul>
</Card>

<Card

#### Validación y reducción de líneas duplicadas en CSV - **fix**

- Optimización del proceso de carga de datos desde archivos CSV.
- Prevención de duplicidades en los ingresos bancarios.

#### Migración de flujo de caja al nuevo dashboard - **migration**

- El módulo se trasladó desde Intranet 1 hacia el dashboard unificado.
- Unifica la visualización financiera bajo el nuevo sistema operativo.

#### Módulo de notificaciones de rescisión de contrato - **feature**

- Generación automática de tres notificaciones para clientes con cartera vencida.
- Pensado para agilizar procesos legales y contractuales.

#### Descarga de reportes de venta - **feature**

- Permite exportar listados de ventas desde el sistema.
- Incluye soporte para filtros activos y formatos ordenados para análisis.

### Portal Cliente

#### Nueva etiqueta visual: Pérdida de ubicación en lote - **feature**

- Se añadió una etiqueta que alerta cuando el cliente ha perdido la asignación de su lote.
- Mejora la transparencia durante el proceso postventa.

### Web Tres Raíces

#### Autocompra en Tres Raíces - **feature**

- Habilita la autogestión de compra para clientes sin intervención del asesor.
- Disponible únicamente en el desarrollo "Tres Raíces" desde App.

## Mejora en gestor de cobranza y listado de ventas nuevas

### Dashboard

#### Listado de ventas nuevas - **feature**

- Módulo para agregar comentarios a ventas en llamadas de éxito.
- Mejora el seguimiento comercial y feedback por parte del equipo.

#### Visualización de gestor de cobranza y flujo de caja - **improvement**

- Visualización más clara del ingreso y conciliación de cuotas.
- Buckets actualizados con nueva categorización.

#### Conciliación de balances de cliente - **fix**

- Permite múltiples reversos sobre el mismo ingreso.
- Simplifica la gestión de correcciones.

#### Almacenamiento con Wasabi - **fix**

- Uso de almacenamiento externo para evitar errores de espacio en disco.

### Portal Cliente

#### Pasarela de pago - Openpay - **feature**

- Integración de Openpay como nuevo proveedor de pagos.
- Mejora la experiencia de pago y seguridad del cliente.

</Update>
