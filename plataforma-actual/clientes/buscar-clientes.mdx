<Steps>
  <Step title="Accede al buscador" icon="solar-system">
    En la parte superior de la vista principal de clientes encontrarás el campo
    de búsqueda:
    <strong>Buscar por nombre o email</strong>
  </Step>
  <Step title="Escribe un valor" icon="pen-to-square">
    Ingresa un nombre completo, parcial o un correo electrónico. El sistema irá
    filtrando los resultados automáticamente.
  </Step>
  <Step title="Consulta los resultados" icon="square-info">
    Los resultados aparecerán directamente en la tabla inferior, mostrando
    nombre, lotes comprados, avance de pago y acciones disponibles.
  </Step>
</Steps>

---

## Funcionamiento del buscador

Al comenzar a escribir, el sistema realiza una búsqueda dinámica en tiempo real, sin necesidad de presionar Enter.

<Tip>
  Puedes usar fragmentos del nombre o del correo electrónico. Por ejemplo:
  escribir <strong>"Us"</strong> mostrará coincidencias como "Usuario ejemplo".
</Tip>

---

## Visualización de resultados

Una vez se detecten coincidencias, la tabla mostrará:

| Campo           | Descripción                                                         |
| --------------- | ------------------------------------------------------------------- |
| Nombre          | Nombre completo del cliente.                                        |
| Lotes comprados | Cantidad de lotes asignados si ya es cliente.                       |
| Avance de pago  | Porcentaje del pago amortizado en caso de tener lotes activos.      |
| Acciones        | Menú de tres puntos para acceder a detalles, ventas o cotizaciones. |

<Note>
  Si el resultado no aparece, asegúrate de escribir correctamente el nombre o
  correo.
</Note>

<Check>
  Puedes combinar la búsqueda con el selector de{" "}
  <strong>“Resultados por página”</strong> para mostrar más clientes si tienes
  una cartera amplia.
</Check>

### Preguntas frecuentes (FAQ)

<AccordionGroup type="multiple">
  <Accordion title="¿Qué datos puedo usar para buscar?">
    Puedes buscar por nombre, apellido o correo electrónico. El sistema identificará coincidencias parciales automáticamente.
  </Accordion>

{" "}

<Accordion title="¿La búsqueda distingue entre mayúsculas y minúsculas?">
  No. El sistema es insensible a mayúsculas, por lo que puedes escribir en
  cualquier formato.
</Accordion>

{" "}

<Accordion title="¿Por qué no aparece un cliente que sí está registrado?">
  Asegúrate de escribir correctamente el nombre o correo. Si sigue sin aparecer,
  verifica en otras vistas como cotizaciones o ventas si se registró con un dato
  diferente.
</Accordion>

{" "}

<Accordion title="¿Puedo buscar por otros campos como teléfono?">
  No por ahora. La búsqueda solo está habilitada por nombre y correo
  electrónico.
</Accordion>

  <Accordion title="¿La búsqueda también aplica a prospectos?">
    Sí. Se incluyen tanto clientes como prospectos en los resultados.
  </Accordion>
</AccordionGroup>
