Desde esta sección puedes visualizar todas las campañas activas, inactivas o expiradas que pueden aplicarse durante el proceso de cotización.  
Estas campañas están disponibles únicamente para el perfil **Asesor**.

---

## 1. ¿Qué información muestra?

Cada campaña se presenta con los siguientes datos:

| Elemento           | Descripción                                                                      |
| ------------------ | -------------------------------------------------------------------------------- |
| Nombre de campaña  | Identificador visible de la promoción.                                           |
| Estado             | Indicado como: <code>ACTIVE</code>, <code>INACTIVE</code>, <code>EXPIRED</code>. |
| Fechas             | Período de vigencia (inicio y fin).                                              |
| Código de campaña  | Se usa en el proceso de cotización para aplicar el beneficio.                    |
| Total de registros | Número de usuarios que han usado el código.                                      |

<Info>
  Los estados como <code>ACTIVE</code> o <code>EXPIRED</code> ayudan a
  identificar rápidamente si una campaña puede ser utilizada.
</Info>

<Card title="Estados posibles" icon="signal">
  <ul>
    <li>
      <code>ACTIVE</code>: Disponible y vigente.
    </li>
    <li>
      <code>INACTIVE</code>: Aún no inicia o está pausada.
    </li>
    <li>
      <code>EXPIRED</code>: Ya finalizó y no puede aplicarse.
    </li>
  </ul>
</Card>

---

## 2. Descargar listado

Cada campaña cuenta con un botón para **descargar el listado de registros en formato Excel (.xlsx)**.

<Info>
  El archivo contiene información de clientes que han usado la campaña:
</Info>

- Nombre y apellidos
- Número de teléfono
- Correo electrónico
- Fecha de registro

<Card title="Formato descargable" icon="file-excel" horizontal={true}>
  El documento está en formato <strong>.xlsx</strong> y es útil para análisis,
  seguimiento o validación de resultados.
</Card>

---

## 3. ¿Para qué sirven las campañas?

Los códigos de campaña pueden utilizarse para:

- Aplicar **descuentos especiales** durante la cotización.
- **Promocionar productos** o etapas específicas.
- Incentivar la compra de lotes durante una temporada determinada.

<Card title="Ejemplo de uso" icon="tag">
  La campaña <strong>“Winter is Comming”</strong> incluye el código{" "}
  <code>DESCUENTO100</code>y permite aplicar una promoción válida entre
  septiembre y diciembre.
</Card>
