Esta sección te permite visualizar y hacer seguimiento a todas las ventas concretadas dentro del sistema, junto con su avance y estatus actual.

---

## 2. Filtros disponibles

En la parte superior de la vista podrás aplicar los siguientes filtros:

- **Rango de fechas**: para ver ventas realizadas en un periodo específico.
- **Búsqueda por cliente o lote**: encuentra rápidamente una venta usando nombre o código de lote.

<Tip>
  Para más detalles, consulta la sección: [Buscar
  ventas](../ventas/buscar-venta)
</Tip>
---

## 3. Tarjetas resumen

Debajo de los filtros, se muestran tarjetas con datos globales:

- **Ventas**: cantidad total de ventas concretadas y su monto acumulado.
- **Apartados**: número de clientes que han pagado el apartado.
- **Enganches**: monto acumulado en enganches.

<Info>
  Estos indicadores se actualizan en tiempo real conforme se registran nuevas
  ventas.
</Info>

---

## 4. Tabla de ventas

La tabla muestra la información detallada de cada venta, incluyendo:

| Campo              | Descripción                                                               |
| ------------------ | ------------------------------------------------------------------------- |
| Fecha              | Fecha en que se concretó la venta.                                        |
| Cliente            | Nombre del cliente involucrado en la transacción.                         |
| Asesor             | Asesor responsable que gestionó la venta.                                 |
| Proyecto / Lote    | Desarrollo, etapa y lote asignado al cliente.                             |
| Monto total        | Monto final de la venta, incluyendo enganche y cuotas programadas.        |
| Avance del proceso | Estatus actual de la venta (ej. Apartado, Contrato firmado, Escriturado). |

<Check>
  Puedes acceder a más acciones desde los **tres puntos** al final de cada fila.
</Check>

<Note>
  Si una venta tiene cuotas vencidas, verás una etiqueta adicional indicando
  cuántas están pendientes.
</Note>

---

## 5. Estatus de una venta

Cada venta puede tener uno de los siguientes estatus, que reflejan su avance en el proceso. A continuación, se describen en detalle:

<Card title="Apartado" icon="bookmark">
  El cliente realizó el pago inicial para reservar su lote. Este es el primer
  paso posterior a la cotización y activa el proceso de venta.
</Card>

<Card title="Documentación" icon="folder-open">
  El cliente está cargando o validando su información y archivos requeridos para
  formalizar su compra.
</Card>

<Card title="Firma en proceso" icon="pen-line">
  El contrato está siendo revisado y está pendiente de firma electrónica. Es una
  etapa crítica antes de formalizar la venta.
</Card>

<Card title="Contrato firmado" icon="file-check">
  El contrato ha sido formalizado exitosamente por el cliente. A partir de este
  momento, inicia el seguimiento postventa.
</Card>

<Card title="Cartera vencida parcial" icon="sensor-on">
  El cliente tiene pagos vencidos, pero aún no suficientes para considerarse en
  morosidad (menos de 4 cuotas vencidas). Es un aviso preventivo.
</Card>

<Card title="Cartera vencida" icon="circle-exclamation">
  El cliente acumula **4 o más cuotas vencidas**. Esta venta está en riesgo de
  cancelación si no se regulariza su situación de pagos.
</Card>

<Note>
  Importante: El sistema indica cuántas cuotas están vencidas para facilitar el
  seguimiento.
</Note>

<Card title="Finalizada" icon="flag-swallowtail">
  El proceso de pago ha concluido satisfactoriamente, pero aún no se ha generado
  la escritura. Esta etapa indica que el cliente ha cumplido con sus
  obligaciones financieras.
</Card>

<Card title="Escriturado" icon="house-flag">
  El cliente ha completado todos los pagos y ya cuenta con las **escrituras
  oficiales** de su propiedad. El proceso de venta ha sido cerrado formalmente.
</Card>
